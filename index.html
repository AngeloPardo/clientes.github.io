¿Qué es la segmentación de clientes?

La segmentación de clientes es el proceso que permite clasificar a cada cliente de Agrocampo según su comportamiento de compra, tipo de actividad y relación comercial. Esta clasificación es la base para el análisis comercial, el diseño de estrategias de venta, la asignación de condiciones y el seguimiento de cada tipo de cliente. Un cliente se clasifica en un único segmento operativo principal, determinado por la combinación de criterios descritos en este documento.

7 segmentos definidos
50 escenarios analizados
6 line discount codes
2 enfoques de criador
Ret · Consumidor final
Uso propio · Bajo volumen · Consumo final
📖 Definición: "Sectores de consumo final = sectores asociados al uso propio del cliente, normalmente para el cuidado, alimentación, bienestar o accesorios de sus mascotas, no para reventa, clínica veterinaria, ganadería, distribución o producción."
Cliente Ret es aquel que compra principalmente sectores de consumo final: alimentos para mascotas, accesorios, snacks, higiene, arenas, juguetes, camas, collares, correas, guacales y productos básicos de bienestar para uso propio del hogar o de sus mascotas.
📌 Criterios teóricos:
  • Compra para uso propio
  • Baja complejidad
  • Bajo volumen
  • Mezcla de producto de consumo final
  • Sin señales fuertes de cliente profesional o negocio
✨ Ejemplo: María Rodríguez
  • Compra 6 bultos de 2kg (12kg total) para sus 2 perros al mes
  • Una sola marca, compra juguete o snack cada 2 meses
  • Desparasitante oral 1 vez al año
  • Volumen mensual: bajo (menos de $200.000 COP)
  • → Clasifica como Ret

Vet · Consultorios / Cliente veterinario
Salud animal · Ldc 12 o 18 · Recurrencia clínica
📖 Definición: "Sectores clínicos veterinarios = sectores asociados a salud animal, tratamiento, farmacia veterinaria, medicamentos, insumos clínicos, material médico, vacunas, antiparasitarios, productos terapéuticos o atención profesional de animales."
Cliente Vet es aquel que Agrocampo reconoce comercialmente como veterinaria mediante line discount code 12 o 18, o aquel que demuestra un comportamiento profesional veterinario por su concentración relevante y recurrente en medicamentos.
📌 Criterios teóricos:
  • Grupo de descuento veterinario 12 o 18
  • Compra recurrente en sectores clínicos veterinarios
  • Alta participación de medicamentos o farmacia veterinaria
  • Variedad técnica de productos y sku
  • Volumen y órdenes suficientes para diferenciarlo de Ret
  • Sin dominancia de concentrado como única señal
✨ Ejemplo: Clínica Veterinaria San Martín
  • Compra semanal: 20 vacunas, 15 antibióticos, 10 antiparasitarios inyectables, material quirúrgico
  • No compra concentrado (las clínicas no venden alimentos)
  • Tiene line discount code 12, compra recurrente todo el año
  • Volumen mensual: más de $2.500.000 COP
  • → Clasifica como Vet

B2B · Distribuidor / B2R / Institucional
Lógica de negocio · Reventa · Acuerdos pactados
📖 Definición: "Cliente B2B es aquel que compra con lógica de negocio, no con lógica de uso propio. Compra para revender, distribuir, surtir un negocio, abastecer una operación, atender clientes finales, cumplir acuerdos comerciales, realizar compras institucionales o manejar operaciones intercompany."
Dentro de B2B se incluyen:
Distribuidor: Cliente que compra para reventa, distribución o abastecimiento mayorista.
B2R: Cliente que compra productos de consumo final, pero no para uso propio sino para venderlos o surtir un negocio.
Institucional: Cliente con acuerdos pactados, ventas institucionales, intercompany, convenios.
📌 Criterios teóricos:
  • Condición comercial de negocio
  • Compra para reventa, distribución o abastecimiento
  • Alto volumen, alta variedad de sku, alta recurrencia
  • Múltiples familias o sectores
  • Acuerdos pactados / intercompany / institucional
  • Sin lógica de uso propio como consumidor final
✨ Ejemplo: Distribuciones El Agro SAS
  • Compra 20 bultos de concentrado de 3 marcas cada 8 días
  • Compra 6 presentaciones de antiparasitarios y medicamentos
  • Ticket promedio: $5.000.000 COP por orden
  • Revende a tiendas de mascotas en toda la región
  • → Clasifica como B2B distribuidor

Gan · Ganadería / Producción pecuaria
Producción animal · Ldc 11,13,14,15,17
📖 Definición: "Cliente Gan = cliente cuya compra está asociada a ganadería, producción animal o actividad pecuaria de campo. Compra productos para bovinos, leche, carne, equinos, porcinos, avícola, finca, sales minerales, suplementos ganaderos, antiparasitarios de grandes animales, vacunas, vitaminas pecuarias, concentrados productivos o insumos de producción."
Cliente Gan es aquel que Agrocampo reconoce comercialmente mediante line discount code 11, 13, 14, 15 o 17, o aquel que demuestra una participación ganadera relevante por mezcla de producto, volumen, cantidades significativas y recurrencia.
📌 Criterios teóricos:
  • Line discount code 11, 13, 14, 15 o 17
  • Alta participación de sectores ganaderos
  • Volúmenes o cantidades significativas
  • Compra asociada a producción pecuaria
  • Productos de bovinos / leche / carne / equinos / pecuarios
  • Sin lógica de consumo final Ret
  • Sin comportamiento clínico Vet como actividad principal
Subcategorías: Gan L (Leche) · Gan C (Carne) · Gan Mixto · Gan Equino · Gan Potencial
✨ Ejemplo: Hacienda La Pradera
  • Compra 9 bultos de concentrado productivo para bovinos cada 15 días (50 cabezas)
  • Compra 30 dosis de vacuna aftosa y 20 antiparasitarios para grandes animales
  • Tiene line discount code 14, volumen mensual: más de $1.800.000 COP
  • → Clasifica como Gan C (carne)

Bre · Criador / Manejo de camadas
Crianza · Varios animales · Volumen superior
📖 Definición: "Cliente Bre = cliente que compra con lógica de crianza, reproducción o mantenimiento recurrente de varios animales, principalmente mascotas. Compra principalmente alimentos, concentrados, nutrición, suplementos, higiene, productos preventivos, desparasitantes, antiparasitarios, productos para cachorros, maternidad, lactancia o cuidado de camadas."
📌 Criterios teóricos:
  • Alta participación en mascotas
  • Alta participación en concentrados / alimentos
  • Recurrencia de compra
  • Volumen superior a consumidor final
  • Productos asociados a crianza, camadas o mantenimiento
  • Sin señal clínica fuerte Vet · Sin señal de reventa B2B
Enfoque 1: Criador de mascotas
  • Perros, gatos, animales de compañía
  • Camadas de cachorros (3-15 por camada)
  • Productos: concentrado para cachorros, vitaminas, maternidad, lactancia
  • Volumen superior a consumidor final
Ejemplo: Criadero Golden "El Encanto"
Compra 6 bultos de concentrado para cachorros/semana, vitaminas y suplementos para 3 camadas. Volumen: +$800.000 COP/mes. → Bre (mascotas)
Enfoque 2: Criador de ganado
  • Bovinos, equinos, animales de producción
  • Cría, levante, reproducción de ganado
  • Productos: concentrados, sales minerales, suplementos, vitaminas pecuarias
  • Volumen alto, orientado a producción
Ejemplo: Hacienda "Los Cedros" (cría bovina)
Compra 12 bultos de concentrado de levante cada 10 días, sales y suplementos para 80 cabezas. Volumen: +$2.000.000 COP/mes. → Gan (producción) o Bre (cría)
Nota: Si es producción (leche, carne, engorde) → Gan. Si es solo cría y reproducción → Bre ganado o Gan potencial.
🔍 Diferencia clave: Bre (mascotas) = cría de perros/gatos · Bre (ganado) = cría de bovinos/equinos · Gan = producción final (leche, carne, engorde)

Mkt · Marketplace / Web / Digital
>59% ventas web · Transversal a todos los segmentos
📖 Definición: "Cliente Mkt = cliente cuya venta está dominada por el sector web. Cliente Mkt es aquel que concentra más del 59% de sus ventas en el sector web, lo que indica una relación comercial principalmente digital con Agrocampo."
📌 Criterios teóricos:
  • Ventas web > 59%
  • Dominancia del sector digital
  • Relación comercial principalmente web
  • No es compra web ocasional
  • Puede cruzarse con Ret, Vet, B2B, Gan o Bre
✨ Ejemplo: Comprador digital
  • 85% de sus compras del último año fueron por la web
  • 15% por call center o tienda física
  • Puede ser Ret, Vet, B2B, Gan o Bre en su comportamiento principal
  • → Se etiqueta como Mkt + [segmento principal]

Ext · Agrocampo exterior / Clientes del exterior
Internacional · Exportación · Intercompany
📖 Definición: "Cliente Ext = cliente del exterior o relacionado con operaciones internacionales, exportación, intercompany o compañías fuera de Colombia. Cliente Ext es aquel que Agrocampo identifica directamente como cliente exterior por código de cliente, razón social, país, operación internacional o marcación comercial."
📌 Criterios teóricos:
  • Cliente del exterior
  • Operación internacional / exportación
  • Intercompany o compañía relacionada
  • Identificación directa por código de cliente
  • No depende del canal ni de la mezcla de producto
✨ Ejemplo: Agroecuador S.A.
  • Razón social con país: Ecuador
  • Código de cliente marcado como exterior
  • Compra para distribución en su país
  • → Clasifica como Ext

Matriz de decisión para clasificar
Sigue estos pasos en orden
1
¿Es cliente del exterior? → Ext
2
¿Más del 59% de sus ventas son por web? → Mkt (se suma al segmento base)
3
¿Compra concentrado con patrón de reventa? → B2B (nunca Ret ni Vet)
4
¿Tiene line discount code 12 o 18 o recurrencia clínica? → Vet
5
¿Tiene line discount code 11,13,14,15,17 o actividad pecuaria? → Gan
6
¿Compra para varios animales con lógica de crianza? → Bre (criador de mascotas o ganado)
7
¿Compra para revender o tiene acuerdos? → B2B
8
Si no aplica nada anterior → Ret
🔔 Reglas clave: El canal no define segmento · Line discount code 99 no clasifica · Una compra no define segmento profesional · No todo medicamento lo convierte en Vet: si es para grandes animales, fortalece Gan
